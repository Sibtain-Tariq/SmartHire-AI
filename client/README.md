# SmartHire AI - Frontend (client)

This folder contains the React + Vite frontend scaffold.

Quick start:

```bash
cd client
npm install
npm run dev
```

This scaffold includes:
- Vite + React
- Tailwind CSS
- React Router
- Axios
- React Hook Form
- Framer Motion
- Lucide Icons

Note: The scaffold includes routing, global theme, and base layout. Pages and components should be added under `src/pages` and `src/components` following the project `PROJECT_STRUCTURE.md`.
